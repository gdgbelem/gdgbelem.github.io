Quer começar a usar o Google Cloud, mas não sabe por onde começar?

Junte-se a nós através do Cloud Study Jam do GDG Belém! Juntos vamos colocar a mão na massa em três laboratórios Google Cloud Platform Essenciais! Você terá acesso gratuito à próxima fase, para que você possa terminar esta série de laboratórios em casa para obter um Google-hosted badge para o seu perfil online. Você poderá obter experiência prática com Docker, Kubernetes e Processamento de Dados Científicos, o que lhe gerará badges adicionais.

Junte-se a este encontro para obter treinamento oficial do Google, com um valor estimado de mais de US$ 70, gratuitamente.

Agenda:
8:30 - Inicio
9:00 - 3 (três) laboratórios do Google Cloud Platform Essentials
11:00 - Coffee Break
11:30 - Introdução ao Docker ou Laboratório de Processamento de Dados Científicos
12:30 - Peguntas
13:00 - Encerramento

Informações Importantes:
1. Confirme presença apenas se tiver certeza que irá participar, pois são poucas vagas;
2. Ao confirmar o RSPV você estará concordando em enviar para Google seus dados de inscrição (nome completo e e-mail);
3. Só será confirmada a inscrição se preenchido o nome completo e o e-mail pelo RSPV;
4. Chegue no horário;
5. Traga um laptop (PC, Mac ou Chromebook)! Os laboratórios serão executados em todas as versões mais recentes dos navegadores populares. Para a melhor experiência, verifique se o seu laptop possui o Firefox ou o Chrome instalado;
6. Venha com a bateria de seu laptop carregada, não sabemos se terá tomada para todos.
